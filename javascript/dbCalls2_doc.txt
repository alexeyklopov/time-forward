---Общее--- 
Примеры работы функций - в dbCalls2Test.js (связан с dbCalls2.html)
			
Если функции в db_calls построены на массивах (из БД запрашивается
строка таблицы как одномерный массив, в ней по номеру элемента ищется
нужное значение), то функции dbCalls2 построены на объектах. Из
БД в объект всё так же передаётся строка, при этом название ключа
совпадает с названием столбца, в котором находится переменная.
Например, пусть есть таблица:
--------------------
| id | name |  time |
--------------------
| 12 | foo  | 01:01 |
| 35 | bar  | 17:05 |
--------------------
Условная функция из db_calls обращается к строке и, найдя строку по id,
выдаёт массив (res = example_of_old_function(id))
res = [12, 'foo', '01:01']
Новая функция выдаст объект
res = Object{id:"12", name:"foo", time:"01:01}
Чтобы узнать время, лежащее в строке, в первом случае следует обратиться
time = res[3]
во втором случае
time = res['time']
Второе удобнее и легче позволяет отловить ошибку. К тому же, если мы
добавим/удалим колонку в таблице, в первом случае всё поедет (а мы об
этом можем и не узнать). Во втором случае всё останется как было.

---Функции---
	1. get_row_by_field(params)
Универсальная функция для доступа к таблицам вида
	usr_[уникальный ID пользователя]_[суффикс таблицы]
	grp_[номер группы]_[суффикс таблицы]
	а также к таблицам users и events_other
ВХОДНЫЕ ДАННЫЕ:
params - объект с полями
	'login': логин пользователя
	'pass': пароль пользователя
	'tableType': 'usr'/'grp'/'users'/'events_other'
	'tableSuffix':суффикс таблицы, например 'lessons'
	'fieldName':имя поля, по которому ищем нужную строку
	'fieldValue': значение в этой строке
ПРИНЦИП ДЕЙСТВИЯ: по введённым данным пользователя определяется его уни-
кальный id (пользователю не виден) или номер группы, после чего из соотв.
таблицы запрашивается строка, в которой поле fieldName имеет значение
fieldValue. Если таких несколько, выводится одна из строк.
ВЫХОДНЫЕ ДАННЫЕ: структура с полями, соответств. полям таблицы
ПРИМЕР: (больше примеров в dbCalls2Test.js)

Входные данные:
	database_params={
			'login':getCookie('login'),
			'pass':getCookie('pass'),
			'tableType':'usr',
			'tableSuffix':'online',
			'fieldName':'id',
			'fieldValue':213,
		};
	var online1 = get_row_by_field(database_params);
	
Выходные данные:
Object { id: "213", day_deadline: "5", min_time: "12", max_time: "12",
 name: "практика по экономике" }
